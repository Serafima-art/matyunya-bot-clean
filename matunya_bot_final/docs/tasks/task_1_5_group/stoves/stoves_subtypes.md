# 🟥 ВОПРОС 3 (стоимость)

## Паттерн stoves_purchase_cost

narratives:

1️⃣ find_price_difference
Определи, на сколько рублей стоимость покупки дровяной печи, которая подходит по объёму парного отделения, меньше стоимости электрической печи с учётом её установки.

2️⃣ find_wood_stove_total_cost
Определи общую стоимость покупки дровяной печи, подходящей по объёму парного отделения, если доставка печи на дачный участок стоит … рублей.

3️⃣ find_electric_stove_total_cost
Определи общую стоимость покупки электрической печи с установкой и доставкой на дачный участок, если доставка стоит … рублей.

4️⃣ find_operating_cost_difference
За год электрическая печь расходует 3100 киловатт-часов (кВт·ч) электроэнергии по цене 4 руб. за 1 кВт·ч, а дровяная печь — 3 кубометра (куб. м) дров по цене 1500 руб. за 1 куб. м.
Определи, на сколько рублей годовые расходы на эксплуатацию дровяной печи, подходящей по отапливаемому объёму парного отделения, меньше расходов на использование электрической печи.

# 🟦 ВОПРОС 5 (радиус арки кожуха)
## Паттерн stoves_arc_radius

narratives:

1️⃣ find_arc_radius
Хозяин выбрал дровяную печь с кожухом вокруг дверцы топки. Верхняя часть кожуха выполнена в виде арки, приваренной к передней стенке печи по дуге окружности с центром в середине нижней части кожуха.
Высота кожуха равна a сантиметров, а его ширина — b сантиметров.
Найди радиус R дуги окружности, образующей арку кожуха.
Ответ дай в сантиметрах.
