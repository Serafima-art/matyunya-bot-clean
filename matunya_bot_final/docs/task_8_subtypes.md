# "Золотой стандарт" подтипов для Задания 8

## 1. Подтип: `power_operations`
### Описание
Чистовые вычисления со степенями, где основание — **число**. Могут быть дроби, скобки, умножение, деление.

### Примеры-оригиналы:
- Найдите значение выражения: `(2⁻⁵ · 2⁻⁶) / 2⁻¹⁵`
- Какое из данных ниже чисел является значением выражения: `(4⁻⁵ · 4⁻⁴) / 4⁻⁸`
- Найдите значение выражения: `( (5² · 5³)⁴ ) / ( (5 · 5⁵)³ )`
- Найдите значение выражения: `( (5²)⁻⁸ ) / 5⁻¹⁵`
- Найдите значение выражения: `(5⁷ · 9⁹) / 45⁸`
- Найдите значение выражения: `1 / (2⁻⁷ · 2⁹)`

---

## 2. Подтип: `variable_power_substitution`
### Описание
Выражение со степенями, где основание — **переменная** (`a`, `b`, `x` и т.д.). Сначала нужно упростить, а потом подставить **числовое** значение.

### Примеры-оригиналы:
- Найдите значение выражения `a⁻⁶ · (a⁵)²` при `a = 5`
- Найдите значение выражения `(a³)^4 / a⁹` при `a = 3`

---

## 3. Подтип: `root_operations`
### Описание
Выражения, для решения которых нужно использовать свойства **корней**. Это может быть умножение, деление корней или формулы сокращенного умножения.

### Примеры-оригиналы:
- Найдите значение выражения: `√17.5⁴ · √17.2²`
- Найдите значение выражения: `(√65 · √13) / √5`
- Найдите значение выражения: `(72 / (2√3)²) `

---

## 4. Подтип: `power_with_root`
### Описание
"Гибридные" задания, где есть **и степени, и корни** одновременно.

### Примеры-оригиналы:
- Найдите значение выражения: `√(16a¹¹) / a⁸` при `a=3`
- Найдите значение выражения: `√(25a¹⁹) / a¹¹` при `a=2`
- Найдите значение выражения: `√(a⁸ · (-a)⁴)` при `a=2`
- Найдите значение выражения: `√(36x⁴ / y²)` при `x=6, y=9`
- Найдите значение выражения: `(√16a⁹ · √4b³) / √(a⁵b³)` при `a=9, b=11`

---

## 5. Подтип: `expression_with_root_substitution`
### Описание
Сложные алгебраические выражения (не только степени), в которые нужно подставить значение с **корнем**.

### Примеры-оригиналы:
- `30a – 5(a+3)²` при `a = √3`
- `-24ab – (-4a + 3b)²` при `а = √7, b = √5`
