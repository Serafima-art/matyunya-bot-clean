# 📌 Как писать ответы в Матюне

✏️ Чтобы не искать специальные математические знаки на клавиатуре, можно писать проще:

## ✅ Корни
• √3 → `r3`  
• 6√3 → `6r3`  
• 1/√3 → `1/r3`  
• ³√5 → `r3(5)`  

## ✅ Степени
• 2³ → `2p3`  
• 5² → `5p2`  
• (−3)² → `(−3)p2`  
• 10⁻¹ → `10p−1`  

---

📌 Запомни легко:  
- **r** = root → корень  
- **p** = power → степень  

✨ Пиши в таком формате, и Матюня всегда поймёт твой ответ!
